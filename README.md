Project Name

New Book Library

Purpose

This is a modern book library web application built with Next.js that allows users to explore, browse, and discover books easily with a clean UI and smooth user experience.

Live URL

https://new-book-laibary.vercel.app

Key Features
Browse books in a modern UI
Search and explore book collections
Beautiful responsive design using Tailwind CSS
Fast performance with Next.js App Router
Theme support using HeroUI
Smooth animations using Animate.css
Fully responsive (mobile + desktop)
